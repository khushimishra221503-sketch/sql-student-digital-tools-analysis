## 💾 SQL Project: Student Digital Tools Analysis

This project analyzes student usage of digital learning tools using SQL.

### 🔧 Tools Used:
- MySQL Workbench
- SQL

### 📊 Key Insights:
- YouTube is the most commonly used learning platform  
- Most students use digital tools for studying  
- Students prefer a mix of digital and traditional learning  
- Major issues include distractions and internet problems
